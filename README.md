# Fishtech Challenge

To run the code locally, after `git clone` the repository, follow the steps below

### Client

1. run `cd client`
2. run `npm install`
3. run `npm start`

### Backend

1. run `cd backend`
2. run `npm install`
3. run `npm start`

Once client and backend are running, go to [http://localhost:3000](http://localhost:3000)
